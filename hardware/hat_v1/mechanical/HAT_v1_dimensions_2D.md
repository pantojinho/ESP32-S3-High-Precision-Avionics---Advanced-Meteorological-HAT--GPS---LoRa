# HAT v1 - 2D Mechanical Drawing Notes

Units: mm

## Base board reference (Waveshare 1.64)

1. Board envelope target: approx. 28.60 x 43.50
2. Standoff holes: 4x M2
3. Header rows: side-aligned through-hole interface

## HAT v1 board target

1. Match base board envelope for bottom stack.
2. Keep all HAT components below standoff height.
3. Preserve access to:
- USB-C
- BAT connector
- RST/BOOT buttons
- SD slot
- RF connector region

## Mandatory clearances

1. Keepout around antenna/feed area.
2. Keepout around button actuators.
3. Keepout around SD insertion path.
