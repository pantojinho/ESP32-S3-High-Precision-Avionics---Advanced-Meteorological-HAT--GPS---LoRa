# HAT v1 Hardware Workspace

This folder stores active hardware artifacts for the bottom HAT board.

## Structure

- `mechanical/`: fit, dimensions, keepouts, and STEP placeholder/export
- `electrical/`: schematic intent and pinout contracts
- `manufacturing/`: BOM and production package notes

## Notes

The active baseline is the Waveshare ESP32-S3 Touch AMOLED 1.64.
